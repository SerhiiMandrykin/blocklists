## Summary

Add malicious domains from CERT-UA#20032 (Кібератаки UAC-0252 з використанням стілерів SHADOWSNIFF та SALATSTEALER).

| Field | Details |
|-------|---------|
| **Threat actor** | UAC-0252 |
| **Campaign** | Since January 2026, UAC-0252 distributes phishing emails impersonating Ukrainian government agencies, tricking victims into downloading malware (SHADOWSNIFF stealer, SALATSTEALER MaaS stealer, DEAFTICK backdoor) disguised as updates for civilian and military mobile applications. |
| **Domains added** | 10 |
| **Source** | https://cert.gov.ua/article/6287707 |

## Domains to block

- `digital-ua.digital`
- `nfkavn.bond`
- `ua-gov.info`
- `ukremail.com`
- `sa1at.ru`
- `salat.cn`
- `salator.es`
- `salator.ru`
- `websalat.top`
- `wrat.in`

## Excluded (legitimate services abused by attackers)

- `github.io` (mk-gov-ua.github.io, govermentnerc.github.io) — GitHub Pages is a legitimate service; blocking github.io would cause widespread collateral damage

## Notes

- UAC-0252 is linked to the "PalachPro" Telegram channel
- Attack chain includes XSS exploitation on vulnerable legitimate websites to deliver malware hosted on GitHub
- Malware families: SHADOWSNIFF (stealer), SALATSTEALER (MaaS stealer), DEAFTICK (Go backdoor), AVANGARD (ransomware)
- Also leverages CVE-2025-8088 (WinRAR exploit)
