## Summary

Add malicious domains from CERT-UA#19092 (UAC-0190 targeting Ukrainian Defense Forces with PLUGGYAPE backdoor via fake charity fund websites).

| Field | Details |
|-------|---------|
| **Threat actor** | UAC-0190 (Void Blizzard / Laundry Bear) |
| **Campaign** | Targeted cyberattacks against Ukrainian Defense Forces using fake charity fund websites to distribute PLUGGYAPE backdoor via messengers. Malware delivered as `.docx.pif` or `.pdf.exe` files in password-protected archives. |
| **Domains added** | 9 |
| **Source** | https://cert.gov.ua/article/6286942 |

## Domains to block

- `hart-hulp-ua.com`
- `harthulp-ua.com`
- `solidarity-help.com`
- `solidarity-help.org`
- `saint-daniel.com`
- `saint-daniel.org`
- `saint-daniel.world`
- `razem-ua.com`
- `frontline-help.com`

## Excluded (legitimate services abused by attackers)

- `pastebin.com` — legitimate paste service used to host C2 configuration
- `rentry.co` — legitimate paste service used to host C2 configuration
- `ghostbin.axel.org` — paste service used to host PLUGGYAPE Python code

## Notes

- UAC-0190 (Void Blizzard / Laundry Bear) has been conducting campaigns from October-December 2025, impersonating charity organizations.
- PLUGGYAPE is a Python-based backdoor using WebSockets and/or MQTT for C2 communication, with JSON data format.
- An upgraded version (PLUGGYAPE.V2) was discovered in December 2025 with MQTT protocol support and anti-analysis checks.
- A Rust-based variant was identified in January 2026.
- Attackers use legitimate Ukrainian phone numbers and Ukrainian language for social engineering via messengers.
