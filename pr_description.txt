## Summary

Add malicious domains from CERT-UA#6288271 (Лікарні, органи місцевого самоврядування та оператори FPV - у фокусі кластера кіберзагроз UAC-0247).

| Field | Details |
|-------|---------|
| **Threat actor** | UAC-0247 |
| **Campaign** | Cyberattacks targeting Ukrainian hospitals, local government bodies, and FPV drone operators via phishing emails disguised as humanitarian aid offers, delivering AGINGFLY RAT, RAVENSHELL, SILENTLOOP, and other malware. |
| **Domains added** | 10 |
| **Source** | https://cert.gov.ua/article/6288271 |

## Domains to block

- `ukrvarta.online` — phishing/malware distribution (fake humanitarian aid site)
- `ukrdopomoga.space` — C2 server (WebSocket)
- `frontforce.org` — fake organization website (AI-generated)
- `nazk.linkpc.net` — malware delivery (DDNS, impersonates NAZK)
- `dsszzi.linkpc.net` — malware delivery (DDNS, impersonates DSSZZI)
- `declaration.linkpc.net` — malware delivery (DDNS)
- `he335f2d353d.publicvm.com` — C2 server (WebSocket, DDNS)
- `edbo.work.gd` — malware delivery (DDNS)
- `buttrocket.work.gd` — malware delivery (DDNS)
- `vizcpalsuvnbhjltpnrxcc5fo7lw95uy7.oast.fun` — C2/exfiltration (OOB interaction)

## Excluded (legitimate services abused by attackers)

- `pastebin.com` — legitimate paste service
- `t.me` — Telegram messenger
- `paste.c-net.org` — legitimate paste service used for C2 config retrieval
- `linkpc.net` (root) — No-IP DDNS service (only attacker subdomains blocked)
- `publicvm.com` (root) — DDNS service (only attacker subdomain blocked)
- `work.gd` (root) — DDNS service (only attacker subdomains blocked)
- `oast.fun` (root) — Interactsh OOB testing service (only attacker subdomain blocked)

## Notes

- UAC-0247 campaign active March-April 2026, intensifying attacks on healthcare (emergency hospitals) and local government
- Attack chain: phishing email → LNK in archive → HTA execution → AGINGFLY RAT deployment
- Also targets Ukrainian Defense Forces (FPV operators) via Signal messenger with trojanized "BACHU" software
- Malware arsenal: AGINGFLY (C# RAT), RAVENSHELL (reverse shell), SILENTLOOP (PowerShell C2 with Telegram fallback), CHROMELEVATOR (credential stealer), ZAPIXDESK (WhatsApp stealer)
- Uses LIGOLO-NG and CHISEL for tunneling, RUSTSCAN for network reconnaissance, XMRIG for cryptomining
